# simpleOcaml
A simple OCaml IDE  
Pour le dark mode, il faudra probablement attendre la v2 (WPF toussa)  
Pour tester l'IDE, clonez le repo et lancez "bin/debug/simpleOCaml.exe" (et si ça marche pas faîtes une issue)
